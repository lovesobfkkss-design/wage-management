# 강한영어수학학원 급여관리시스템

## 제품 개요

학원 운영에 특화된 웹 기반 급여 관리 시스템입니다. 시급제 직원(조교, 파트강사)과 비율제 강사의 급여를 통합 관리하며, Firebase를 통한 실시간 데이터 동기화로 여러 기기에서 동시 접속이 가능합니다.

## 타겟 사용자

- **관리자**: 학원 원장 또는 급여 담당자
- **직원**: 조교, 파트강사, 비율제 강사

## 핵심 기능

### 1. 다중 사업장 지원
- 여러 학원/지점을 하나의 시스템에서 관리
- 사업장별 직원 필터링

### 2. 시급제 직원 관리
- **조교**: 고용보험 0.8% 공제
- **파트강사**: 사업소득세 3.3% 공제
- 2단계 시급제 지원 (시간대별 차등 시급)
- 반올림 규칙 설정 (정각/30분/시간 단위)

### 3. 비율제 강사 관리
- 강사별 수수료 비율 설정 (50%, 60% 등)
- 월별 학생 명단 및 수강료 관리
- CSV 파일로 학생 데이터 일괄 등록
- 카드수수료 1% 자동 계산

### 4. 근무 기록
- 출퇴근 시간 기록
- 휴게시간 관리
- 월별 근무 내역 조회

### 5. 급여 정산
- 자동 급여 계산 (시급제/비율제)
- 공제 내역 상세 표시
- CSV 내보내기

### 6. 문자 생성
- 급여 안내 문자 템플릿 자동 생성
- 복사하여 SMS 전송 가능

### 7. 실시간 동기화
- Firebase Realtime Database 연동
- 여러 기기에서 동시 접속/수정 가능
- 오프라인 시 로컬 캐시 활용

## 사용 사례

1. **월말 급여 정산**: 직원별 근무시간 확인 → 급여 계산 → CSV 내보내기 → 문자 발송
2. **일일 출퇴근**: 직원이 앱에서 본인 이름 선택 후 출근 기록
3. **비율제 강사 관리**: 학생 명단 입력 → 수강료 합계 → 강사 몫 자동 계산

## 기술적 특징

- 별도 서버 설치 불필요 (Firebase 사용)
- 모바일 반응형 디자인
- 브라우저에서 바로 실행 가능

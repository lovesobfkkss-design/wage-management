# SPEC-PDF-001 수용 기준

## 1. 기능 테스트 시나리오

### TC-001: 시급제 직원 PDF 생성

**Given** 관리자가 급여정산 탭에 있고, 2026년 1월 데이터가 있을 때
**When** 시급제 직원(예: 박태균)의 "명세서" 버튼을 클릭하면
**Then** 다음을 확인:
- [ ] PDF 파일이 자동 다운로드됨
- [ ] 파일명: `급여명세서_박태균_2026년1월.pdf`
- [ ] PDF 내용에 회사명 "강한영어수학학원" 포함
- [ ] PDF 내용에 직원 이름, 직급 표시
- [ ] PDF 내용에 근무시간, 시급, 세전급여 표시
- [ ] PDF 내용에 공제 유형, 공제율, 공제액 표시
- [ ] PDF 내용에 실지급액 표시
- [ ] 한글이 정상 표시됨

---

### TC-002: 비율제 강사 PDF 생성

**Given** 관리자가 급여정산 탭에 있고, 비율제 강사 데이터가 있을 때
**When** 비율제 강사의 "명세서" 버튼을 클릭하면
**Then** 다음을 확인:
- [ ] PDF 파일이 자동 다운로드됨
- [ ] 파일명: `급여명세서_[강사이름]_[YYYY년MM월].pdf`
- [ ] PDF 내용에 회사명 "강한영어수학학원" 포함
- [ ] PDF 내용에 강사 이름, 정산비율 표시
- [ ] PDF 내용에 총 수강료, 카드수수료 표시
- [ ] PDF 내용에 강사 정산액 표시
- [ ] PDF 내용에 사업소득세(3.3%) 표시
- [ ] PDF 내용에 실지급액 표시

---

### TC-003: 근무 기록 없는 직원

**Given** 관리자가 급여정산 탭에 있고, 특정 월에 근무 기록이 없는 직원이 있을 때
**When** 해당 직원의 정산 내역을 조회하면
**Then** 다음 중 하나를 확인:
- [ ] "명세서" 버튼이 비활성화되거나 숨겨짐
- [ ] 또는 버튼 클릭 시 "근무 기록이 없습니다" 알림 표시

---

### TC-004: 학생 없는 비율제 강사

**Given** 관리자가 급여정산 탭에 있고, 특정 월에 학생이 없는 강사가 있을 때
**When** 해당 강사의 정산 내역을 조회하면
**Then** 다음 중 하나를 확인:
- [ ] "명세서" 버튼이 비활성화되거나 숨겨짐
- [ ] 또는 버튼 클릭 시 "정산 대상이 없습니다" 알림 표시

---

## 2. 비기능 테스트 시나리오

### TC-005: PDF 생성 속도

**Given** 정상적인 데이터가 있을 때
**When** PDF 생성 버튼을 클릭하면
**Then** 다음을 확인:
- [ ] PDF 다운로드가 3초 이내에 시작됨
- [ ] 브라우저가 멈추거나 느려지지 않음

---

### TC-006: 크로스 브라우저 호환성

**Given** PDF 파일이 생성되었을 때
**When** 다음 브라우저에서 PDF를 열면
**Then** 다음을 확인:
- [ ] Chrome: 정상 열림
- [ ] Edge: 정상 열림
- [ ] Firefox: 정상 열림 (선택)
- [ ] Safari: 정상 열림 (선택)

---

### TC-007: 기존 기능 영향 없음

**Given** PDF 기능이 추가된 후
**When** 기존 CSV 내보내기 버튼을 클릭하면
**Then** 다음을 확인:
- [ ] CSV 파일이 정상 다운로드됨
- [ ] CSV 내용이 이전과 동일함

---

## 3. 엣지 케이스 테스트

### TC-008: 특수문자 이름

**Given** 직원 이름에 특수문자가 포함되어 있을 때 (예: 김(테스트))
**When** PDF를 생성하면
**Then** 다음을 확인:
- [ ] PDF가 정상 생성됨
- [ ] 파일명에서 특수문자 처리됨
- [ ] PDF 내용에서 이름이 정상 표시됨

---

### TC-009: 큰 금액 표시

**Given** 정산 금액이 천만원 이상일 때
**When** PDF를 생성하면
**Then** 다음을 확인:
- [ ] 금액이 천단위 콤마로 표시됨 (예: 10,000,000)
- [ ] 레이아웃이 깨지지 않음

---

### TC-010: 긴 근무 기록

**Given** 한 달에 30일 이상 근무 기록이 있을 때
**When** PDF를 생성하면
**Then** 다음을 확인:
- [ ] PDF가 정상 생성됨
- [ ] 필요시 페이지 분할 처리됨

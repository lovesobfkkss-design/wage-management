# SPEC-PDF-001: 급여 명세서 PDF 출력

## 1. 개요

### 1.1 목적
급여관리시스템에서 직원별 급여 명세서를 PDF 형식으로 다운로드할 수 있는 기능을 제공합니다.

### 1.2 범위
- 시급제 직원(조교, 파트강사) 급여 명세서 PDF 생성
- 비율제 강사 정산 명세서 PDF 생성
- 브라우저 기반 클라이언트 사이드 PDF 생성

### 1.3 용어 정의
- **급여 명세서**: 직원의 근무 내역, 급여 계산, 공제 내역을 포함한 공식 문서
- **jsPDF**: 브라우저에서 PDF를 생성하는 JavaScript 라이브러리

---

## 2. 요구사항 (EARS 형식)

### 2.1 기능 요구사항

#### R1: 시급제 직원 PDF 생성
**GIVEN** 관리자가 급여정산 탭에서 시급제 직원의 정산 내역을 조회하고 있을 때
**WHEN** "명세서" 버튼을 클릭하면
**THEN** 해당 직원의 급여 명세서가 PDF 파일로 다운로드된다

#### R2: 비율제 강사 PDF 생성
**GIVEN** 관리자가 급여정산 탭에서 비율제 강사의 정산 내역을 조회하고 있을 때
**WHEN** "명세서" 버튼을 클릭하면
**THEN** 해당 강사의 정산 명세서가 PDF 파일로 다운로드된다

#### R3: PDF 내용
**WHEN** PDF가 생성될 때
**THEN** 다음 정보가 포함되어야 한다:
- 회사명 (강한영어수학학원)
- 발급일
- 정산월
- 직원/강사 정보 (이름, 유형)
- 계산 내역 (근무시간/수강료, 시급/비율, 세전급여)
- 공제 내역 (공제 유형, 공제율, 공제액)
- 실지급액

#### R4: 파일명 규칙
**WHEN** PDF 파일이 다운로드될 때
**THEN** 파일명은 `급여명세서_[이름]_[YYYY년MM월].pdf` 형식이어야 한다

### 2.2 비기능 요구사항

#### R5: 성능
**THE SYSTEM SHALL** PDF 생성을 3초 이내에 완료해야 한다

#### R6: 호환성
**THE SYSTEM SHALL** 생성된 PDF가 Chrome, Edge, Firefox, Safari에서 정상 열리도록 해야 한다

#### R7: 한글 지원
**THE SYSTEM SHALL** PDF에서 한글 텍스트가 정상 표시되어야 한다

#### R8: 오프라인 동작
**THE SYSTEM SHALL** 인터넷 연결 없이도 PDF 생성이 가능해야 한다 (CDN 캐시 후)

## 3. 기술 스택

### 3.1 라이브러리 선택
- **jsPDF 2.5.1**: 브라우저 기반 PDF 생성
- CDN 방식 로드

### 3.2 선택 이유
- 서버 불필요 (클라이언트 사이드)
- 경량 (~150KB)
- 한글 지원
- MIT 라이선스 (무료)

## 4. 제약사항

### 4.1 기술적 제약
- 기존 바닐라 JavaScript 아키텍처 유지
- Firebase와의 충돌 방지
- 기존 CSV 내보내기 기능 유지

### 4.2 비즈니스 제약
- 회사명은 하드코딩 (강한영어수학학원)
- 서명란은 빈칸으로 유지 (수기 서명용)

# 비율제 강사 시스템 추가 계획

## 현재 상황
- 시급제 조교 (고용보험 0.8% 공제)
- 시급제 파트강사 (사업소득세 3.3% 공제)
- 곽병학, 유향순 선생님 데이터 존재 (삭제 예정)

## 추가할 기능

### 1. 비율제 강사 관리 (새 탭)
**위치:** 관리자 메뉴에 "비율제 강사" 탭 추가

**강사 등록 정보:**
- 이름
- 비율 설정 (예: 50%, 60%, 70% 등 - 강사가 가져가는 비율)
- 공제 항목: 카드수수료 1% + 사업소득세 3.3%

### 2. 학생 관리 (비율제 강사별)
**기능:**
- 강사 선택 후 Excel 업로드 (학생명, 수강료)
- 또는 직접 학생 추가/수정/삭제
- 월별로 학생 목록 관리

**Excel 형식:**
| 학생명 | 수강료 |
|--------|--------|
| 홍길동 | 300000 |
| 김철수 | 250000 |

### 3. 비율제 강사 정산 계산
**계산 순서:**
1. 총 수강료 합계
2. 카드 수수료 1% 공제 (전체 수강료에서)
3. 강사 비율 적용 (예: 50%면 나머지 금액의 50%)
4. 사업소득세 3.3% 공제 (강사 몫에서)
5. 실지급액 산출

**예시 (5:5 강사, 총 수강료 100만원):**
```
총 수강료: 1,000,000원
카드수수료 (1%): -10,000원
수수료 공제 후: 990,000원
강사 몫 (50%): 495,000원
사업소득세 (3.3%): -16,335원
실지급액: 478,665원
```

### 4. 기존 시스템 변경
- 곽병학, 유향순 기본 데이터에서 삭제
- 직원 유형 구분 명확화:
  - 조교 (시급제, 0.8%)
  - 파트강사 (시급제, 3.3%)
  - 비율제 강사 (수강료 비율, 1% + 3.3%)

### 5. UI 구조

**관리자 메뉴:**
```
대시보드 | 직원관리 | 비율제강사 | 근무기록 | 급여정산 | 문자생성 | 설정
```

**비율제강사 탭 화면:**
1. 강사 목록 (추가/수정/삭제)
2. 강사 선택 → 학생 관리
3. 월별 정산 내역

**급여정산 탭:**
- 기존 시급제 정산 + 비율제 강사 정산 통합 표시

## 확인 필요 사항

1. ✅ 비율은 강사마다 다르게 설정 (50%, 60% 등)
2. ✅ 카드수수료 1%는 전체 수강료에서 먼저 공제
3. ✅ 학생 Excel은 강사별로 따로 업로드
4. ❓ 한 학생이 여러 강사에게 수업 받는 경우 있나요?
5. ❓ 수강료는 월 단위로 관리하나요?
